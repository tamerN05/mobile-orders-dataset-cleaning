# mobile-orders-dataset-cleaning

## Project overview 
This project focus on cleaning a raw dataset and preparing it for analysis and decision-making using Microsoft Excel.
## data cleaning steps
- removed duplicates
- fixed innisistent text values
- split columns using text to columns
- handled missing values
- standardized data , number and currency formats

## Tools used
-Microsoft Excel

## Final Results
- The dataset is now clean , structured , and ready for analysis.
